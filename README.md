# Mecanizados UPC

Este repositorio contiene una versión analizada y reconstruida de la página principal de **Mecanizados UPC**, basada en la extracción de estilos, colores y estructura visual del sitio original.

El objetivo es estudiar, documentar y mejorar la arquitectura visual, la paleta de colores y la organización del contenido para futuras propuestas de rediseño o auditorías técnicas.

## 🛠️ Objetivo del proyecto

- Documentar la identidad visual real del sitio.
- Analizar la coherencia de colores, tipografías y jerarquías.
- Preparar una base para un rediseño más moderno, claro y profesional.
- Facilitar pruebas, auditorías y comparativas de diseño.
